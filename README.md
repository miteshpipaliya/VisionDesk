📚 VisionBook – Smart Sticky Notes Productivity App (V1)

VisionBook is a modern macOS-inspired sticky notes productivity web application designed for students, developers, and productivity enthusiasts.

It transforms your browser into a digital desktop workspace where you can freely create, move, resize, and organize sticky notes just like on macOS.

The application also includes:

🗒️ Sticky Notes

🧾 To-Do Lists

📓 Personal Journal

🌤️ Live Weather Widget

🕒 Digital Clock

🎨 Customizable Notes

VisionBook focuses on delivering a minimal, aesthetic, and smooth productivity experience.

📌 Introduction

VisionBook is built to simulate a desktop-like workspace inside the browser.

Instead of static notes, VisionBook provides interactive sticky notes that behave like macOS desktop notes:

Drag anywhere

Resize dynamically

Stack notes

Customize fonts and colors

Smooth animations

The goal is to create a beautiful digital thinking workspace where users can:

Capture ideas

Study efficiently

Organize thoughts

Track tasks

Write journals

All inside a smooth macOS-inspired interface.
